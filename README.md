# attendance system

# requirements
RaspberryPi 3 or 4

NFC reader RC-S380

# installation
```bash
sudo pip install nfcpy
```

# run
```python
sudo python main.py
```

# note
You need to run as root.

# device info
- username: attendance
- password: attendance
- devicename: raspberrypi

# ssh
```bash
ssh attendance@raspberrypi.local
```
